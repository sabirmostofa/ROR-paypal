PayPal Ruby on Rails SDK

PayPal Ruby on Rails SDK for NVP API

The PayPal's Software Developer Kit (SDK) eases the process of integrating PayPal's financial services into your application by providing a small footprint of three files caller.rb, profile.rb, and utils.rb. These three files are developed as a plug-in. These files are located under PayPalSDK/lib folder which is under Vendor/plugins folder. The plug-in supports only 3-token credentials at this time. The SDK comes with web samples written for Rails which illustrates how to use PayPal NVP Web Services API including examples for Direct Payment, Express Checkout, TransactionSearch, Refund, Void, and Capture. The Sample application does not use a database hence it is not a typical Rails application.

The latest upgrades, FAQs and announcements can always be found at: Tools for PayPal Developers

Installation

1) Unzip the file for example to c:\
2) On command prompt, cd to c:\PayPal_Ruby_SDK
3) Run the command rails server
     You can see the folllowing once the server has started:
     Rails application started on http://0.0.0.0:3000
4) Open a browser and type http://IP ADDRESS of your machine:3000 on the address bar and hit return button.

Configuration for using certificate

1) In "vendor\plugins\payal_sdk\lib\caller.rb" file change http.verify_mode=OpenSSL::SSL::VERIFY_NONE to http.verify_mode=OpenSSL::SSL::VERIFY_PEER
2) Uncomment the following lines of code
   #cert = File.read("sdk-seller_cert_key.pem")
   #http.ca_file = 'api_cert_chain.crt'
   #http.cert = OpenSSL::X509::Certificate.new(cert)
   #http.key = OpenSSL::PKey::RSA.new(cert)
   #http.cert.verify(http.key)
3) In "vendor\plugins\payal_sdk\lib\profile.rb" use the credentials for certificate and change the endpoint to "api.sandbox.paypal.com"

Proxy Support

Proxy support has been added for environments behind an HTTP proxy server.
If the calls need to be made via a proxy sever, set USE_PROXY flag to true,
ADDRES to proxy server IP address, and PORT to port number of the server in @proxy_info hash in the profile class.
